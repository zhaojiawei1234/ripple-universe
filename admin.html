🔐 修改管理密码

当前密码通过环境变量 RIPPLE_ADMIN_PASSWORD 设置，默认值为 ripple2025。
修改方法：重启服务器时设置环境变量，例如：

# Windows PowerShell:
$env:RIPPLE_ADMIN_PASSWORD="你的新密码"; node server.js

# Windows CMD:
set RIPPLE_ADMIN_PASSWORD=你的新密码 && node server.js

# Linux/Mac:
RIPPLE_ADMIN_PASSWORD=你的新密码 node server.js
